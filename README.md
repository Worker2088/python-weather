# Python Weather

Веб-приложение для отслеживания погоды в выбранных локациях, построенное на Django. Пользователи могут регистрироваться, искать города и добавлять их в свой личный список для быстрого доступа к текущим погодным данным.

## Основные возможности

- **Управление пользователями**: Регистрация, вход и выход. Используется кастомная модель пользователя.
- **Поиск локаций**: Интеграция с OpenWeather API для поиска городов по названию.
- **Избранные локации**: Возможность сохранять города в профиль и просматривать погоду для всех выбранных мест на главной странице.
- **Погодные данные**: Текущая температура, состояние неба, влажность и другие параметры.
- **Кэширование**: Использование Redis для оптимизации запросов к API и ускорения работы приложения.
- **Контейнеризация**: Полная поддержка Docker и Docker Compose для быстрого развертывания.

## Стек технологий

- **Язык**: Python 3.12
- **Фреймворк**: Django 6.0+
- **База данных**: PostgreSQL
- **Кэширование**: Redis
- **Менеджер пакетов**: [uv](https://github.com/astral-sh/uv)
- **Интеграция**: OpenWeather API
- **Тестирование**: Pytest, Pytest-django
- **Линтинг и типизация**: Ruff, Mypy
- **Развертывание**: Docker, Docker Compose


## Структура проекта

- `siteweather/` — Основная директория Django проекта.
  - `locations/` — Приложение для управления локациями и погодой.
  - `users/` — Приложение для управления пользователями (кастомная модель).
  - `core/` — Общие вспомогательные модули и исключения.
  - `settings/` — Настройки проекта (base, local, production, test).
- `static/` — Статические файлы (CSS, изображения).
- `templates/` — Общие HTML шаблоны.
- `tests/` — Тесты проекта.
